# Engineer Ali Al-Kelabi Portfolio

A responsive personal portfolio website showcasing my background in AI, robotics, automation, and data engineering.

## Overview

This project is a professional portfolio website designed to present my technical background, selected projects, experience, education, and contact information in a clean and modern format. It was built to support job applications and to provide a clear overview of my practical engineering work.

## Features

- Responsive layout for desktop and mobile devices.
- Light and dark theme support.
- Hero section with professional positioning.
- Profile section with work style, availability, and key strengths.
- Technical skills grouped by category.
- Selected projects section with project summaries and technologies used.
- Experience timeline.
- Education section.
- Downloadable CV link and contact links.
- Smooth section reveal animations.
- Mobile navigation menu.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Responsive design principles
- CSS variables
- Intersection Observer API
